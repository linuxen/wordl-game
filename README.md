
# Wordle Console Game

Консольное приложение для игры в **Wordle** на C++.  
Программа выбирает случайное слово заданной длины из словаря и даёт пользователю 6 попыток его отгадать.

---

## Возможности

- Чтение словаря из файла, переданного аргументом командной строки
- Выбор случайного слова длины `N`
- 5 попыток на отгадывание
- Подсветка статуса каждой буквы:
  - `+` — буква есть и стоит на своей позиции
  - `=` — буква есть в слове, но на другой позиции
  - `-` — буквы нет в загаданном слове
- Корректная обработка ошибок словаря и ввода

---

## Запуск

Приложение принимает **два позиционных аргумента**:

```bash
./app PATH N
````

Где:

* `PATH` — путь к файлу словаря
* `N` — длина загаданного слова (целое число)

Примеры:

```bash
./app dictionary.txt 5
./app ./data/words_en.txt 6
```

Можно считать, что программа всегда запускается с корректными аргументами по типу (путь существует, `N` — положительное целое).

---

## Формат словаря

* Один слово **на строку**
* Максимальная длина слова — **100 символов**
* Разрешены **только буквы** (без цифр, пробелов, знаков препинания и т.п.)

Пример файла `dictionary.txt`:

```text
hello
world
hatch
coach
kyoto
spell
llama
```

Если в словаре **нет ни одного слова длины `N`**, программа:

* выводит сообщение об ошибке в консоль
* завершает работу с  `There are no words of such length`

---

## Правила игры

1. Из словаря случайным образом выбирается слово длины `N`.
2. Пользователю даётся **5 попыток** отгадать слово.
3. На каждой попытке пользователь вводит слово длины `N`.
4. Программа выводит строку статусов из `+`, `=`, `-` той же длины:

   * `+` — буква есть и стоит на том же месте
   * `=` — буква есть в слове, но в другой позиции
   * `-` — буквы нет в загаданном слове
5. Если пользователь угадывает слово полностью (все `+`), программа печатает:

   ```text
   WINNER!!!
   ```

   и завершает игру.
6. Если за 6 попыток слово не угадано, программа выводит:

   ```text
   The word was: ***
   ```

   где `***` — загаданное слово.

## Пример игровой сессии

Допустим, приложение запущено так:

```bash
./app dictionary.txt 5
```

и случайно загадано слово `hello`.

Игровая сессия может выглядеть так:

1. Игрок вводит: `hatch`
   Ответ:

   ```text
   +---=
   ```

2. Игрок вводит: `coach`
   Ответ:

   ```text
   -=--=
   ```

3. Игрок вводит: `kyoto`
   Ответ:

   ```text
   --=-+
   ```

4. Игрок вводит: `lllll`
   Ответ:

   ```text
   ==++=
   ```

5. Игрок вводит: `spell`
   Ответ:

   ```text
   --=+=
   ```

6. Игрок вводит: `hello`
   Ответ:

   ```text
   +++++
   WINNER!!!
   ```

---

## Обработка ввода

* Длина введённого слова не должна превышать **100 символов**

* Пользователь сам отвечает за то, что вводит осмысленные слова — приложение не проверяет их наличие в словаре.

## Технические детали

* Язык: **C++** (стандарт не ниже C++11 / C++17 — в зависимости от реализации)
* Для генерации случайных чисел используется стандартная функция `std::rand`.
